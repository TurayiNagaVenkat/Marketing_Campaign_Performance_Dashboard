# Marketing_Campaign_Performance_Dashboard
This project analyzes marketing campaign data to evaluate campaign performance,
user conversions, and revenue contribution using Power BI.

## Key Insights
- Campaign-wise performance comparison
- Conversion rate analysis
- Revenue contribution by source
- Time-based performance trends

## Tools Used
- Power BI
- DAX
- Data Cleaning & Modeling
